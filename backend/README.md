## Description

A sample demonstrating serving a VueJS website from a NestJS backend.

## Installation

```bash
$ npm install
# Install Vue dependencies
$ cd client && npm i && cd ..
$ npm run build --prefix client
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Endpoints

### /api/v1/users/create

Method: POST
Description: Creates a user. If the user exists returns 201 as well.
Body:
```json
{
  "userId": "358154856046788609",
  "username": "username",
  "avatar": "testAvatarString",
  "verified": true
}
```

### /api/v1/servers

Method: GET
Description: Get a list of all servers

### /api/v1/servers/create

Method: POST
Description: Create a new server
Body:
```json
{
  "owner": "358154856046788609",
  "description": "blubserver"
}
```
